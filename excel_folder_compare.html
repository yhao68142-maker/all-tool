Excel F栏与文件夹名称比对工具

功能：读取 Excel 指定列（默认 F 栏）中 Q / T / R / S / M 开头的数据，自动展开像 SO2AS0001110~SO2AS0001113 这样的编号范围， 再和你选择的目录中的文件夹名称进行比对，输出缺少哪一项。

1. 参数与文件

支持 .xlsx / .xls。默认读取第一张表，你也可以手动切换。
默认 F。也可以改成 A / B / C ...
只处理这些字母开头的数据。大小写都可以。
推荐按钮可直接读取所有子文件夹名称（Chrome / Edge 效果最好）。
兼容模式会读取目录中的文件路径推断文件夹名称，但空文件夹可能无法识别。
